HTML FORM TAG

this tag is used to get information about the user or details about the user

FORM TAGS

  1. Fieldset tag - creates the border ine around the form or form elements
  2. Legend tag - it is the caption or title for the fieldset tag
  3. input -> 15+ - text, number, email, password, radio, checkboxes, date, datetime, datetime-local, time, month, week, file, search, color, tel(deprecvated), range, submit, hidden, reset, image, url
  4. Label - label for input box
  5. select - dropdown single, multiple
  6. textarea - multi line input
  7. button -> submit, reset, button
