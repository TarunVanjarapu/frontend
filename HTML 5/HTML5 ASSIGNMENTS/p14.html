Curriculum Overview

Week 1-2: HTML & CSS Basics

Learn how to structure and style web pages using HTML and CSS.

  • Introduction to HTML: Tags, Elements, Structure
  • CSS Basics: Selectors, Properties, and Styling
  • Building Simple Web Pages
  • Responsive Web Design with Media Queries

Week 3-4: JavaScript Fundamentals

Understand the core concepts of JavaScript to make your web pages interactive.

  • JavaScript Syntax and Variables
  • Functions, Loops, and Conditional Statements
  • DOM Manipulation and Event Handling
  • Working with Arrays and Objects

Week 5-6: React & Modern Web Apps

Build dynamic and scalable web apps using React.js, a powerful JavaScript library.

  • Introduction to React: Components and JSX
  • State and Props
  • Event Handling and Forms in React
  • Building Single Page Applications (SPAs) with React

Final Project: Build & Deploy a Portfolio

Create your personal portfolio and deploy it online for the world to see!

  • Designing a Portfolio Layout
  • Integrating React Components
  • Deploying the Website on GitHub Pages or Netlify
  • Project Presentation & Reflection
